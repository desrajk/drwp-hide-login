== Description ==
Secure your WordPress site by hiding the default login page. Set a custom login slug, restrict by IP, and send secure temporary login URLs to administrators.

== Features ==
* Custom login URL
* Blocks access to wp-login.php
* IP whitelisting
* Temporary token login URLs
* Admin bar shortcuts

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/drwp-hide-login` directory.
2. Activate the plugin.
3. Go to **Settings > General** to configure your login slug and allowed IPs.
